MyBatis 搭建

IDEA Java project

在IDEA中创建空项目，在其设置中找到"Build, Execution, Deployment"->“Maven”

修改其中的几个路径

  • “Maven home path” 可改为自己安装的，如"C:/Program Files/apache-maven-3.9.9"
  • “User setting file” 改为Maven路径下的配置文件，如“C:\Program Files\apache-maven-3.9.9\conf\settings.xml”
  • “Local repository” 可改为自己的仓库路径
